# Contract Analysis Chatbot

## Overview
This project builds an AI-powered chatbot capable of analyzing and answering questions from contract documents using Natural Language Processing (NLP) and Large Language Models (LLMs).

The system processes legal text, extracts relevant information, and enables users to query contracts conversationally.

## Features
- Contract document processing
- Natural language question answering
- Information extraction from legal text
- Retrieval-based document understanding

## Methodology
1. Text preprocessing and document parsing
2. Embedding generation for contract sections
3. Semantic retrieval of relevant clauses
4. LLM-powered question answering

## Technologies Used
- Python
- NLP
- LangChain
- LLMs
- Vector databases / embeddings

## Use Cases
- Contract review assistance
- Legal document analysis
- Clause extraction
- Automated document querying

## Disclaimer
This project is for educational purposes and should not replace professional legal advice.
